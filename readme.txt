System Requirements
===================

Software Requirements
---------------------
The DBH is supported on the following operating systems 
   
   Linux
   Windows 7
   Windows 10 

Hardware Requirements/Recommendations
-------------------------------------
   Intel Core i3 2100 or later for Windows 
   Color display capable of 1024 X 768 pixel resolution
   

Memory Requirements/Recommendations
-------------------------------------
DBH  requires approximately
200 M of available disk space on the drive or partition.

DBH requires a minimum of
2G of RAM for clustering 16S rRNA dataset.


User's Guide
=================

Input Sequences Requirements
----------------------------
Input file requires fasta or fastq format

Execute Step
------------
Step 1: compile codes using make command
Step 2: ./db then come out options reuired
Step 3: The final clustering results are presented in text formats (*.clstr_index) and can be found in the input file directory.
